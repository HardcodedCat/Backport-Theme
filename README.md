# Backport Theme

An theme for backporting latest Discord's fonts and colors for the Aliucord [Themer](https://github.com/Vendicated/AliucordPlugins/tree/main/Themer) plug-in

<img src="screenshots.png"/>

#### Used Fonts
- GGSans ExtraBold
- GGSans Bold
- GGSans SemiBold
- GGSans Medium
- GGSans Normal
- SourceCodePro Semibold
